# ❄️ Confetti Studio

A premium, interactive confetti experience built with modern web technologies. This project transforms a simple button into a professional "Confetti Studio" with multiple high-quality visual effects, glassmorphism UI, and optimized performance.

## ✨ Features

- **Ice Burst**: A crisp, cyan and white celebration themed like ice.
- **Classic Party**: The timeless multicolor explosion.
- **Side Cannons**: Double-sided wide spread fireworks.
- **Realistic Physics**: Multi-layered particles with realistic gravity and decay.
- **Night Sky**: A continuous firework simulation (Start/Stop).
- **School Pride**: Elegant gold and red burst (customizable).
- **Premium UI**: Glassmorphism dashboard with a sleek dark aesthetic.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.

## 🛠️ Built With

- **HTML5 & CSS3**: Custom styles with glassmorphism and modern typography.
- **JavaScript (Vanilla)**: Optimized logic for interactive effects.
- **[Canvas-Confetti](https://github.com/catdad/canvas-confetti)**: The high-performance library powering our festive animations.
- **Google Fonts (Outfit)**: Premium typography for a modern look.
- **Font Awesome**: High-quality icons for an intuitive interface.

## 🚀 Getting Started

### Prerequisites

No installation is required! This project runs entirely in the browser.

### Usage

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.
3. Click on any of the effect cards to trigger a celebration!
